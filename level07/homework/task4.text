HTML ნიშნავს Hypertext Markup Language-ს. 

Hypertext - ნიშნავს "ჰიპერთექსტს", ანუ ტექსტს, რომელიც შეიცავს ბმულებს (ლინკებს) სხვა დოკუმენტებზე ან ვებგვერდებზე გადასასვლელად. მაგალითად, როდესაც ვებგვერდზე დააწკაპუნებ ლინკზე და გადადიხარ სხვა გვერდზე, ეს სწორედ ჰიპერთექსტის წყალობით ხდება.
Markup - ნიშნავს "მარკირების ენას", რაც ნიშნავს, რომ HTML გამოიყენება დოკუმენტის სტრუქტურის განსაზღვრად (მაგალითად, რა არის სათაური, აბზაცი, სურათი და ა.შ.).
Language - ეს უბრალოდ ნიშნავს, რომ HTML არის პროგრამირების ენა, რომელიც გამოიყენება ვებგვერდების შესაქმნელად.
Hyperlink არის ტექსტი ან სურათი, რომელზეც დაწკაპუნებით მომხმარებელი გადადის სხვა ვებგვერდზე, დოკუმენტზე ან რესურსზე. ჰიპერბმულები ხდიან ინტერნეტს "ქსელად", რადგან ისინი აკავშირებენ სხვადასხვა ვებგვერდებს ერთმანეთთან.
მოკლედ რომ ვთქვათ, ინტერნეტი ვერ იარსებებდა ჰიპერბმულების გარეშე, რადგან ის არის მისი მთავარი "ჩონჩხი"
